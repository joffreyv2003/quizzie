Informatie over Minecraft

Minecraft is een sandbox-game gemaakt door Markus "Notch" Persson. Het werd later ontwikkeld door zijn bedrijf Mojang. Het spel werd uitgebracht in 2011.

In Minecraft kan je veel verschillende dingen doen. Het spel is blok-achtig, waarmee dus wordt bedoeld dat alles in de game kubus vormen heeft. Het spel genereert automatisch een omgeving (of wereld genoemd) met grondstoffen, tegenstanders, dieren, grot- en tunnelsystemen, bergen en meren. Iedere speler in Minecraft heeft zijn eigen "karakter". Ieder karakter kan aangepast worden hoe de speler het wilt.

Wat uniek aan Minecraft is, is dat het beide offline en online gespeeld kan worden. Minecraft zit vol met Servers die aangemaakt zijn door spelers. Iedereen die Minecraft speelt, kan een eigen server maken als ze dat willen. Ze kunnen deze server downloaden of kopen via de officiÃ«le Minecraft website. Iedere server heeft zijn eigen regels, mocht je deze dus verbreken heeft de eigenaar van de server er volledig recht op om jou te verbannen van hun gemaakte server.
